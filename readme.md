# Weather Chile App

Aplicación frontend de clima desarrollada con HTML5, Bootstrap 5, JavaScript y SASS.

La aplicación muestra información climática de distintas ciudades y zonas de Chile como Santiago, Valparaíso, Arica, Iquique, Temuco, Chiloé y Punta Arenas.

## Tecnologías utilizadas

- HTML5
- Bootstrap 5
- JavaScript
- SASS (SCSS)

## Metodología de estilos

Se utiliza metodología BEM para organizar las clases CSS.

Ejemplos:

- `.weather-app__title`
- `.place-card`
- `.place-card__temp`

## Funcionalidades actuales

- Listado de localidades
- Vista detalle del clima
